# 🏎️ F1 Tracker

iOS app for tracking Formula 1 — teams, grands prix, standings.

Inspired by **Box Box**, built with SwiftUI & Liquid Glass (iOS 26+).

## Features

- 🏁 Race schedule & live results
- 🏎️ Constructor & driver standings
- 📊 Session data (practice, qualifying, sprint, race)
- 🌙 Dark-first design, car-oriented, colorful minimalism

## Tech Stack

- Swift 6 / SwiftUI
- iOS 26+ (Liquid Glass)
- Jolpica F1 API
- Xcode Cloud CI/CD
